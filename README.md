**Fe_Helper 小助手**
## 使用场景 ##

 - 开发过程中经常需要不同权限的账号切换
 
## 主要功能 ##

 - 账号切换
 - 支持添加线上、线下账号
 - 预览机切换，切换状态无需手动刷新页面
 
 ## 解决的问题 ##
 - 切换账号麻烦
 - 预览机完事之后忘记清楚状态
 - 预览机麻烦，只需在当前页面切换状态，无需手动刷新
